Hi 你好呀，朋友。这是一个实验环境，又或特化的知识图谱，是一个用于思考的被系统标记后的数据结构。笔记系统使用【KCMS】创建，本体使用Obsidian写作，采用quartz框架发布，所有内容都将不断地被推理、证伪、重建、修改，是一个不断生长的笔记系统。与此共同进步的还有经过编程与数理思维设计后的KCMS。如果你觉得尚可，也许可以点亮一颗小星星以资鼓励。

每个条目第一行为当前条目的语义空间定义。

npx quartz build --serve

npx quartz update
